    Diorama

    For my first Blender assignment, I created a diorama. A diorama is like a tiny model that shows a scene with 3D figures. It can be small or big, like something you'd see in a museum.
    The assignment was tough because I was still learning Blender, the software we use to make 3D models. We had to start with basic shapes and build from there. Even though it was challenging, I eventually finished it, and I'm happy with how it turned out.
    I really like working in Blender. It's fun to create things in 3D, even if it takes a lot of practice to get it right.
